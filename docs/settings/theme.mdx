# テーマの変更

アプリケーションのテーマを変更することができます。<br></br>

<Image src="/assets/setting/setting-personal.png" alt="Setting-Personal" height="500" />

「テーマを変更する」をタップするとダイアログが表示され、次の3つの選択肢があります。

<Image src="/assets/setting/theme/theme-dialog.png" alt="Theme" height="500" />

- システム
    - 端末の設定情報から使用するテーマを決定します。
- ライトテーマ
    - 明るいテーマを利用します。
- ダークテーマ
    - 暗いテーマを利用します。

<Info>テーマの設定は、端末ごとに行うことができるので、スタッフのお好みで設定してください。</Info>
