# 患者の詳細

この画面では患者個人の状態を表示することができます。<br></br>

## 詳細

画面上部の「詳細」タブをタップする(初期表示)と患者の詳細画面を表示することができます。
表示されているものは以下の通りです。

<Image src="/assets/monitoring/monitoring-patient-detail1.png" alt="Monitoring-Patient-Detail-1" height="500" />

<Image src="/assets/monitoring/monitoring-patient-detail2.png" alt="Monitoring-Patient-Detail-2" height="500" />

<Image src="/assets/monitoring/monitoring-patient-detail3.png" alt="Monitoring-Patient-Detail-3" height="500" />

1. 基本情報
2. オプションセンサー(別売りのオプションセンサーを使用している場合)
3. 患者の重心位置
4. 患者の体重のグラフ
5. 患者の心拍数と呼吸数のグラフ(別売りのオプションセンサーを使用している場合)

## アラートログ

画面上部の「アラートログ」タブをタップすると患者のアラートログを表示することができます。
